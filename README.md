# 🚀 Central de Monitoramento de Missões Espaciais

Aplicativo mobile desenvolvido em React Native + Expo para simular uma central de monitoramento espacial em tempo real.

---

# 📱 Funcionalidades

- Dashboard em tempo real
- Monitoramento de sensores espaciais
- Sistema automático de alertas críticos
- Atualização manual da missão
- Persistência local com AsyncStorage
- Navegação entre telas com Expo Router
- Gerenciamento global de estado com Context API
- Formulários com validação

---

# 🛰️ Tecnologias Utilizadas

- React Native
- Expo
- Expo Router
- TypeScript
- Context API
- AsyncStorage
- React Hook Form

---

# 🎨 Tema Visual

O aplicativo foi desenvolvido com uma interface futurista inspirada em centrais espaciais da NASA e SpaceX, utilizando:
